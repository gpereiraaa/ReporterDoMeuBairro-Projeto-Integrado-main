## Objetivos deste Projeto

Criar um sistema que, de maneira eficiente e rápida, realize o processo de comunicação dos cidadãos com a administração pública sobre problemas presentes na infraestrutura e manutenção.

## Justificativa do Projeto

O sistema foi criado com o objetivo de facilitar a forma de reportar um problema urbano através dos moradores de determinada cidade, para que a administração pública solucione os problemas.
Exemplo: Um morador de Jandira observa um buraco na rua em que mora. O usuário do sistema “Repórter do Meu Bairro” notifica este problema, a informação chega até os órgãos públicos e o problema é resolvido.

## Descrição do Projeto e Principais Requisitos
Esse projeto visa criar um sistema do zero, que facilite aos cidadãos registrar os problemas urbanos em seu bairro ou cidade. O projeto será desenvolvido da seguinte forma, utilizando o Banco de dados MySQL, Back-End desenvolvido utilizando Node.js e Front-End utilizando HTML, CSS e JavaScript.

## Requisitos Principais

* Registrar novas ocorrências com:

  * Descrição

  * Categoria do problema

  * Localização

  * Anexar fotos

  * Informar dados de contato (opcional)

* Visualizar ocorrências, com possibilidade de:

  * Consultar por região ou cidade

  * Filtrar por status

  * Ver detalhes da ocorrência

  * Acompanhar status das ocorrências:

  * Visualizar status das próprias ocorrências

  * Receber notificações sobre atualizações

* (Opcional) Implementações adicionais:

  * Sistema de autenticação

  * Comentários em ocorrências

  * Avaliação da resolução

  * Painel administrativo para órgãos responsáveis
